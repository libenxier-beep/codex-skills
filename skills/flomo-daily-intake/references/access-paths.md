# Flomo Access Paths

Use the most structured available source. Do not invent memos when no source is available.

## Preferred Order

1. User-provided export, pasted memo list, or file attachment.
2. Configured Flomo API, CLI, MCP, or local connector.
3. Browser or web scraping only when the user explicitly authorizes that path.

## Required Evidence

For each memo, preserve as many of these as the source provides:

- timestamp
- source ID or stable memo URL
- tags
- raw text
- media or transcript status

## Blocker Report

If no usable access path exists, stop with:

- requested scope
- attempted access path
- exact blocker
- whether user-provided export or connector setup would unblock the task
