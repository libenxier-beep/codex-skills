# Example 01: 小红书选题到 Shot List

## User Input

```text
这个选题想做小红书图文：把碎片灵感变成一篇能发布的笔记。先别生成图，给我配图方案。
```

## Expected Read Path

- `SKILL.md`
- `references/style-dna.md`
- `references/bluegrid-ip.md`
- `references/composition-patterns.md`

## Expected Output

给 3 张图即可，不直接调用图像生成。

| 图 | 主题 | 核心意思 | 结构类型 | 小蓝格动作 | 元素 | 标注词 |
| --- | --- | --- | --- | --- | --- | --- |
| 1 | 灵感降噪 | 灵感不是越多越好，先听出重点 | 碎片收拢 | 小蓝格戴耳机听一团噪音线，从里面拉出一条蓝线 | 噪音线团、耳机、蓝线、空白笔记页 | 灵感 / 噪音 / 重点 / 先听清 |
| 2 | 结构上架 | 把素材放进结构，内容才可发布 | 方法分层 | 小蓝格把纸片推进一个蓝色收纳格 | 素材纸片、收纳格、标签、出口门 | 素材 / 结构 / 可发布 / 别堆满 |
| 3 | 真感受出口 | 笔记需要一个真实感受作为出口 | 卡点纠偏 | 小蓝格守住出口，挡掉空泛总结，只放行真感受 | 出口门、红色退回标、笔记卡片 | 空话退回 / 真感受 / 发布 |

## Acceptance Notes

- 没有生图。
- 每张图只讲一个动作。
- 小蓝格不是站在旁边，而是执行动作。
- 标注词少而短，适合放进生图提示词。
