# 小蓝格 IP

## 角色定义

小蓝格是这个 skill 的固定视觉 IP。

它不是吉祥物、贴纸或装饰，而是一个安静、可靠、略带干活感的内容系统操作员。每张图里，小蓝格都要承担画面的核心动作。

## 外形

- 天蓝色小圆角方块。
- 轮廓略微手绘，不要完美几何。
- 两个黑色小圆点眼睛，表情冷静、认真。
- 细线胳膊和细线腿，动作简单。
- 固定配件：细线蓝色过耳式耳机。
- 耳机要轻巧、少细节、可识别；不要麦克风、不要大耳罩、不要 DJ 感。
- 身体填色可以有轻微手绘水彩感和边缘深浅变化，但不要变成精致 3D、渐变图标或商业矢量。
- 身体可以轻微变形成抽屉、便签、模块、网格块、收纳格，但仍要认得出“小蓝格”。

## 性格

- 安静、认真、可靠。
- 像一个低调的内容整理员。
- 有一点冷幽默，但不卖萌。
- 擅长把混乱东西放进结构里。
- 不抢戏，但离开它画面就失去核心动作。

## 常见职责

让小蓝格承担核心动作：

- 推抽屉，把素材放进结构。
- 戴着耳机听噪音，从里面挑出重点。
- 把碎片贴到一张笔记上。
- 拧动“判断”旋钮。
- 拉一条蓝线，把散点收拢。
- 守住发布出口，挡掉堆满的内容。
- 把一句话压成一个小标题。
- 在架子里分拣想法、素材、金句。
- 给内容贴小标签。
- 把混乱线团绕成一条路径。
- 从一堆截图里捞出一个真感受。
- 把多个工具格子拼成一个工作台。

## 配件规则

耳机是识别锚点，不是主题。

- 默认深蓝或钴蓝细线。
- 保持轻、少、清楚。
- 可以随动作轻微倾斜。
- 不要变成头盔、猫耳、耳麦、主播设备或赛博装备。
- 如果耳机被其他物件遮住，要至少露出一侧耳罩或弧线。

## 禁止

- 不要画成儿童卡通。
- 不要过度可爱、圆润、Q 版。
- 不要复杂服装、表情包、闪亮眼睛。
- 不要给它加帽子、围巾、背包等第二配件，除非用户明确要求。
- 不要让它只是站在角落看。
- 不要让它变成普通蓝色方块，耳机必须可见。
- 不要把它画成科技机器人、App icon 或企业 logo。
- 不要把它画成圆润可爱的蓝色玩偶、Q 版角色、儿童贴纸或表情包。

## 判断标准

如果去掉小蓝格，图的核心隐喻还能完全成立，说明小蓝格太装饰了。重写提示词，让小蓝格成为动作主体。
