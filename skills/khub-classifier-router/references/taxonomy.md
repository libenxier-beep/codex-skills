# KHub Taxonomy

## Primary Note Types

- `seed`: early signal, partial thought, still forming
- `cognition`: reusable judgment lens or framework
- `concept`: stable atom, definition, principle, or model
- `viewpoint`: a viewpoint or stance with explicit angle
- `playbook`: process, checklist, or operating sequence
- `prediction`: future-facing claim or expected outcome

## Reject State

- `reject_for_now`: raw signal exists but should not enter KHub yet

## Mapping Principle

- Prefer the smallest stable note type that does not overstate maturity.
- If the item is still exploratory, bias toward `seed`.
- If the item mainly changes how future problems will be judged, bias toward `cognition`.
- If the item is a reusable named idea that can stand alone, bias toward `concept`.
